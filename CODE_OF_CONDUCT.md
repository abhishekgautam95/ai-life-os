# Code of Conduct

## Our Standard

We want AI Life OS to be a respectful, practical, and constructive open-source project.

Contributors are expected to:

- communicate respectfully
- give feedback in a clear and professional way
- assume good intent
- focus on improving the product and codebase
- avoid harassment, abuse, or hostile behavior

## Unacceptable Behavior

This project does not accept:

- harassment or intimidation
- discriminatory language or attacks
- personal insults
- bad-faith arguments
- deliberate disruption of community discussion
- publishing private information without permission

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, pull requests, or contributions that violate this code of conduct.

In serious or repeated cases, contributors may be temporarily or permanently blocked from participating.

## Reporting

If you experience or observe unacceptable behavior, open a private report through the repository security or contact channel when available.

## Scope

This code of conduct applies to:

- GitHub issues
- pull requests
- code review discussions
- project-related communication spaces
